◆ _cuphyStateUpdateType

enum _cuphyStateUpdateType

cuPHY static and quasi-static state update types

Enumerator
CUPHY_STATE_UPDATE_TYPE_CREATE_STATIC 

To be used only at creation time. Result: (a) Memory allocation for static state (b) State initialization.

CUPHY_STATE_UPDATE_TYPE_CREATE_QUASI_STATIC 

To be used only at creation time. Result: (a) Memory allocation for quasi-static state (b) Quasi-static state initialization.

CUPHY_STATE_UPDATE_TYPE_MODIFY 

To be used only during (re)configuration and is relevant only to quasi-static state. Results in state update only (no memory allocation).

CUPHY_STATE_UPDATE_TYPE_DESTROY 

Cleanup allocated memory and state handles.
