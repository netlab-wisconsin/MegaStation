uint8_t** _cuphyPdschDataIn::pTbInput

array of transport block input buffers, one buffer per cell, indexed by cellPrmDynIdx. Each pTbInput[] element points to a flat array with all TBs for that cell. Currently per-cell TB allocations are contiguous, zero-padded to byte boundary. When a cell is in test mode, then the buffer contains the PN23 (pseudorandom sequence) rather than a payload.
