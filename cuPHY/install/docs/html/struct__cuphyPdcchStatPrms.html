uint16_t _cuphyPdcchStatPrms::nMaxCellsPerSlot

◆ pOutInfo

cuphyTracker_t* _cuphyPdcchStatPrms::pOutInfo

pointer to cuphyTracker_t. Its pMemoryFootprint pointer will be updated by cuPHY in channel objection creation. The caller will thus have access to the cuphyMemoryFootprint object of that cuPHY PDCCH channel object. The cuphyMemoryFootprint object tracks the size of the GPU memory allocations owned by that cuPHY PDCCH channel object.
